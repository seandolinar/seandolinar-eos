# seandolinar-eos
Demo EOSIO block reading project. (https://github.com/EOSIO/eos)


#### install packages
`npm install`

#### run the webpack dev server
`npm run start:dev`

#### build the bundle js for deploy
`npm run start:build`

#### build and server ups the app on express
`npm run start:build:serve`

#### run the tests
`npm run test`
